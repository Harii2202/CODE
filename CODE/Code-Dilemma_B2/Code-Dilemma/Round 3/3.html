Programming Questions

1. Reverse a String

Q1: How do you take a string input from the user?

Q2: How do you initialize an empty string to hold the reversed version?

Q3: How do you iterate through the original string backwards?

Q4: How do you build the reversed string one character at a time?

Q5: How do you display the reversed string?

Pseudocode:

START
    INPUT string
    SET reversed = ""
    FOR each character in string from last to first
        ADD character to reversed
    ENDFOR
    PRINT reversed
END
