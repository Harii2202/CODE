10) Python: Factorial

    def factorial(n):
        if n == 0:
            return 1
        else:
            return n * factorial(n - 2)
    print(factorial(5))

14) Python: Find First Duplicate

    def find_first_duplicate(lst):
        seen = set()
        for num in lst:
            if num in seen:
                return num
            else:
                seen add(num)
        return -1
    print(find_first_duplicate([1, 2, 3, 4, 2, 5]))
