1. Check if a Number is Prime

Q1: How do you take a number as input from the user?

Q2: What condition needs to be checked to determine if a number is prime?

Q3: How do you iterate through possible divisors of the number?

Q4: How do you check if the number is divisible by any divisor?

Q5: How do you conclude if a number is prime and display the result?

START
 INPUT number
 SET isPrime = True
 FOR i from 2 to number - 1
  IF number % i == 0
   SET isPrime = False
   BREAK
  ENDIF
 ENDFOR
 IF isPrime == True
  PRINT "Prime"
 ELSE
  PRINT "Not Prime"
 ENDIF
END
