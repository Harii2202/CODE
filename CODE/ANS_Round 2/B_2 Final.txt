14) Python: Find First Duplicate</h2>
                <pre>
    def find_first_duplicate(lst):
        seen = set()
        for num in lst:
            if num in seen:
                return num
            else:
                seen add(num)
        return -1
    print(find_first_duplicate([1, 2, 3, 4, 2, 5]))
Answer: seen.add(num)
